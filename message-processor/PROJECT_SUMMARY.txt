PACS.009.001.08 PAYMENT MESSAGE PARSER
=====================================

Complete Java Project for Parsing ISO 20022 pacs.009 XML Messages
and Converting to Internal Simplified Format

ARCHITECTURE
════════════

Input Format: pacs.009.001.08
  │
  ├─→ Validation (ConversionValidationResult)
  │
  ├─→ Parsing (DOM-based XML parsing)
  │     - Group Header extraction
  │     - Transaction iteration
  │     - Element-level extraction
  │
  ├─→ Mapping (Type conversions)
  │     - ISO 20022 enums → simplified enums
  │     - Complex types → flat models
  │     - Optional fields → null-safe
  │
  ├─→ Conversion (Pacs009ToSimplifiedConverter)
  │     - Batch header
  │     - Transactions (parties, accounts, amounts)
  │     - Remittance & tax information
  │     - Processing metadata
  │
  └─→ Output Format: Simplified Payment v1
       - Clean XML structure
       - Easy to process downstream

USAGE EXAMPLES
══════════════

Command Line:
  java -jar pacs009-parser.jar input.xml output.xml

Programmatic:
  Pacs009ToSimplifiedConverter converter = new Pacs009ToSimplifiedConverter();
  SimplifiedPaymentBatch batch = converter.convert(xmlString);
  
  for (SimplifiedTransaction txn : batch.getTransactions()) {
      System.out.println("Amount: " + txn.getAmount());
      System.out.println("Debtor: " + txn.getDebtor().getName());
      System.out.println("Creditor: " + txn.getCreditor().getName());
  }
