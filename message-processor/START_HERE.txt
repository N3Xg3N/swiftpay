🚀 QUICK START (3 STEPS)
════════════════════════

Step 1: Enter the project directory
    cd pacs009-parser

Step 2: Build with Maven
    mvn clean package
    
    This creates:
    - target/pacs009-parser-1.0.0-SNAPSHOT-shaded.jar (executable JAR)

Step 3: Run the converter
    java -jar target/pacs009-parser-1.0.0-SNAPSHOT-shaded.jar \
         src/test/resources/samples/sample-pacs009.xml \
         output.xml

That's it! You now have converted pacs.009 to simplified format.
